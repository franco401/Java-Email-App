# Java-Email-App
A full stack, email application using Java in the backend with Spring Boot and JavaScript in the frontend with React.

# Versions Used
* Spring Boot 3.3.0
* Java 22

# Spring Boot Dependencies
* Lombok
* Spring Web
* PostgreSQL Driver
* Apache Commons Codec 1.17

# React Dependencies
``` npm install react ```

``` npm install react-router-dom ```

``` npm install axios ```
